# Powerlifting Assistant — Android (Jetpack Compose)

Клиентская часть приложения «Ассистент пауэрлифтера».

## Что реализовано
- Firebase Auth (email/password)
- Главная: карточки (калории, ИМТ, достижения), календарь тренировок, быстрые действия
- Программа: генерация на сервере и просмотр ближайших тренировок
- Восстановление: опросник + рекомендация с сервера, старт тренировки
- Тренировка: таймер + завершение (фиксируется длительность)
- Питание: дневник калорий/белка, цели, добавление/удаление приёмов пищи
- Достижения: список + добавление/удаление (в MVP без фото)
- Калькулятор веса штанги (блины)

## Настройка

### 1) Firebase
Замените файл:
- `app/google-services.json`

Сейчас в репозитории лежит **заглушка**, чтобы проект собирался. Для реальной авторизации нужен ваш Firebase проект.

### 2) Базовый URL сервера
По умолчанию:
- `http://10.0.2.2:8080/` (Android Emulator → localhost компьютера)

Можно переопределить через Gradle property:
```bash
./gradlew :app:assembleDebug -PPOWERLIFT_SERVER_BASE_URL="https://<your-neon-or-server>/"
```

### 3) Cleartext HTTP
В `AndroidManifest.xml` включено `android:usesCleartextTraffic="true"`, чтобы локально ходить на `http://10.0.2.2:8080`.
